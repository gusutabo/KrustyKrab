# KrustyKrab

[README - English](./README.md)

> [!IMPORTANT]  
> Este projeto é bem pequeno, está em desenvolvimento e não é responsivo (e talvez nunca seja, porque não sei se vou mexer nele novamente).

🍔 Um site inspirado no Siri Cascudo de Bob Esponja.

## Funcionalidades

- Página inicial com apresentação do restaurante  
- Página de cardápio com itens de comida  
- Formulário de reservas  
- Formulário de contato  

## Como usar

### Clonar o repositório

```bash
git clone https://github.com/gusutabo/KrustyKrab.git
```

### Abrir o projeto

Basta abrir o arquivo `index.html` no navegador.

## Objetivo

O objetivo deste projeto é praticar HTML e CSS construindo um site temático simples com múltiplas páginas e uma estrutura básica de layout.
